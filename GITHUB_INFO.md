## About Section
A feature-rich online bookstore built with MongoDB, Express.js, React.js, and Node.js. Includes user authentication, shopping cart, multiple payment options (COD, Card, UPI, NetBanking), order tracking, receipt generation, and comprehensive admin panel for managing books and orders.
